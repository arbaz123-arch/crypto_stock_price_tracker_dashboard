Crypto & Stock Dashboard:-

This project is a simple, responsive dashboard built using HTML, CSS, and JavaScript. It allows users to search and track both cryptocurrencies and stocks, view real-time market data, check price charts, and save favorites. The design is clean, beginner-friendly, and suitable for portfolios or practical projects.


Features:-

Search and view data for cryptocurrencies and stocks
Live price, market cap, volume, and percentage change
Seven-day price chart using Chart.js
Add and remove favorites (saved in localStorage)
USD to INR currency converter
Light and dark mode
Mobile-friendly responsive layout


Technologies Used:-

HTML, CSS, JavaScript
Chart.js for charts
Public APIs for crypto, stocks, and currency data

Folder Structure:-
crypto-dashboard/
│── components/   # Header and footer
│── pages/        # Dashboard and about page
│── css/          # Stylesheets
│── js/           # Logic, APIs, utils
│── images/       # Logo or assets


Deployment:-

Can be deployed easily on GitHub Pages for the frontend.
Backend or proxy support can be added later on platforms like Render for extended stock APIs or secure keys.
